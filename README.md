## About

**Aphelion** is a dark, atmospheric 3D RPG/Adventure game built in Unity 6.

Set in a world where light is a dying resource, the player navigates through environments designed to evoke dread, wonder, and discovery. The project doubles as a living programming guide — 55 chapters of development notes covering everything from basic C# scripting to professional-level Unity architecture.

---

## Features

- **Atmospheric 3D environments** — custom shaders, fog, and post-processing
- **RPG mechanics** — inventory, stats, interaction system
- **Custom ShaderLab shaders** — hand-written GPU shaders for visual effects
- **55-chapter dev guide** — full Unity 6 programming documentation included
- **Unity 6 architecture** — built using the latest Unity LTS patterns

## Project Structure

```
Assets/
├── Scripts/            # C# game logic
├── Scenes/             # Unity scenes
├── Shaders/            # Custom ShaderLab shaders
├── Art/                # 3D models, textures, VFX
└── UI/                 # Interface elements

ProjectSettings/        # Unity project configuration
Packages/               # Unity package dependencies
```

## Getting Started

### Prerequisites

- [Unity 6](https://unity.com/releases/unity-6) (LTS recommended)
- Git LFS (for large assets)

### Steps

```bash
git clone https://github.com/JoshTVR/Aphelion.git
```

Open the project folder in **Unity Hub** → select Unity 6 editor → open.
